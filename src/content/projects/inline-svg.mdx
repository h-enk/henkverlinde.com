Inline SVG is the official inline SVG integration for the [Thulite web framework](/projects/thulite).

## Features

- Shortcode for the [SVG](https://developer.mozilla.org/en-US/docs/Web/SVG) element
- Partial for the SVG element
- Store your SVGs as a page resource or global resource
- Add SVGs that respect dark mode and light mode
- Add icons from any icon set like for example [Tabler Icons](https://tabler.io/icons) or [Feather Icons](https://feathericons.com/)
- Specify SVG attributes like `width`, `height`, `viewBox`, and more

## Icon sets

You can install and use _multiple_ icon sets. Change the default icon set in your `params.toml` file.

```toml title="params.toml"
# Inline SVG (@thulite/inline-svg)
[inline_svg]
  iconSetDir = "tabler-icons" # "tabler-icons" (default)
```
