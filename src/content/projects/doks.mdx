**Doks** is a full-featured documentation theme built on the Thulite framework, designed to help developers create fast, accessible, and SEO-optimized documentation websites with minimal setup. It includes features like search, dark/light mode, code highlighting, internationalization, and versioning support — all aimed at giving users a polished reading experience.

With Doks, you get ready-to-use templates, clear site structure, and seamless integration with the Thulite ecosystem, making it easy to build, customize, and deploy professional documentation sites for projects of any size. Whether you're documenting APIs, guides, or product manuals, Doks delivers speed, clarity, and flexibility out of the box.
