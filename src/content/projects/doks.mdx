Doks is a full-featured documentation theme for the [Thulite web framework](https://thulite.io/).

## Features

- Minimal layout.
- Easy-to-read typography.
- Powerful search.
- Supports dark mode and/or light mode.
- Code highlighting.
- Blog pages.
- Internationalization support.
- Versioning support.
- Fully customizable.
- Easy to use and extend.
- Developer friendly.
- Community driven.

## Thulite powered

- Fully optimized for speed.
- Built-in security features.
- SEO friendly.
- Solid and flexible architecture.

## Thulite integrations

- Images
- SEO
- Inline SVG
