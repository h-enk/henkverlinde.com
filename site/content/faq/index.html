What license are your tools under?

All my tools are released under the MIT License. In short, you can do whatever you want with my tools – modify them, rename them, sell them – as long as the end result contains the original copyright notice and permission notice.

Do your tools come with a warranty?

No. They are provided "as is", without warranty of any kind.

What changes can I make to your tools?

Any changes you want. The only requirement is that your modified version also contains the original copyright notice and permission notice.

Do you offer support on your tools?

No. I do not offer support on my tools. However, each tool comes with extensive documentation that should be sufficient to help you out.

Where can I report an issue?

Most of my tools are available on GitHub. If that's the case, create a new bug report issue in the repository of the tool. Otherwise, send me an email.

How do I propose an enhancement?

Most of my tools are available on GitHub. If that's the case, create a new feature request issue in the repository of the tool. Otherwise, send me an email.

How do I submit a change to one of your tools?

Most of my tools are available on GitHub. If that's the case, create a new pull request in the repository of the tool. Otherwise, send me an email.

Do you have another question? Send me an email.
